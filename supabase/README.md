## Supabase setup (copy/paste)

### 1) Run schema + RLS

- Open Supabase Dashboard → **SQL Editor**
- Paste the contents of `supabase/SETUP_SQL.sql`
- Run it

### 2) Auth settings

In Supabase Dashboard → **Authentication**:

- **Providers**: Email enabled
- **Email confirmations**: OFF (so signup logs in immediately)
